# Memo App

Tech Stack:

- Javascript, ES6
- React 16.9
- Ant Design
- Node.js, Express Server

Deployed at Heroku: [https://memo.mrhmt.com/](memo.mrhmt.com)

Run with Docker Compose:

```bash
$ docker-compose up
```

Run with Node.js and NPM:

```bash
$ npm install
$ npm run build && npm run start
```
